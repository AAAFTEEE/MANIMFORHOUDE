## Manim⚙️
Manim 是一个用于创建数学动画的 Python 库，旨在将抽象的概念转化为直观的可视化内容。

### 核心特性
- **数学友好**：原生支持 LaTeX 公式渲染。
- **脚本驱动**：通过 Python 代码定义场景与动画逻辑。
- **高质量输出**：可生成视频、GIF 或图片序列，支持高清渲染。

### 典型应用场景

-   **STEM 教学视频制作**：为数学、物理、计算机科学等学科的抽象概念（如微积分、线性代数、算法）制作解释性动画。
-   **学术演示**：在学术报告、论文或在线文章中嵌入动态图解，以更好地阐述复杂原理或过程。
-   **算法与数据结构可视化**：清晰展示排序、搜索、图遍历等算法的执行步骤。
-   **数据与模型演示**：动态展示统计过程、机器学习模型的决策边界演变等。
